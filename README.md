# saraGongora_Ag14_DSI

Descrição do Projeto

Este projeto consiste no desenvolvimento de uma página web estática utilizando HTML5 e CSS3 com o objetivo de apresentar informações e imagens sobre atrativos turísticos da cidade de Votorantim-SP.

A aplicação foi desenvolvida com foco em organização visual, responsividade e estruturação semântica do conteúdo, permitindo a exibição dos pontos turísticos em formato de colunas com imagens ilustrativas e descrições informativas.

Objetivos
Aplicar conceitos básicos de desenvolvimento web;
Utilizar estrutura HTML semântica;
Trabalhar estilização com CSS;
Organizar elementos em layout de colunas;
Inserir imagens e textos informativos;
Desenvolver uma interface visual simples e responsiva.
Tecnologias Utilizadas
HTML5
CSS3

Funcionalidades
Exibição de seis atrativos turísticos;
Layout em duas colunas;
Responsividade para dispositivos móveis;
Inserção de imagens locais;
Estrutura visual com cartões (cards);
Rodapé contendo referências das imagens.
Responsividade

O projeto utiliza CSS Grid e Media Queries para adaptar automaticamente o layout em diferentes tamanhos de tela.

Computadores: 2 colunas;
Dispositivos móveis: 1 coluna.
